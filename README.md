# FinRL

#### 介绍
量化交易+深度强化学习项目。使用DRL智能体进行投资决策，最大化累计奖励值。实现算法有DQN\DDQN\DDPG\TD3\SAC\PPO\REDQ-DDPG\REDQ-SAC\OFENet等模型。交易市场为美股道琼斯工业指数市场。任务包含交易任务和资产分配任务。
